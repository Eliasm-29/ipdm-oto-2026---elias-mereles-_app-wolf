# Woof App
 
Trabajo práctico de la materia **Dispositivos Móviles** — IPDM Otoño 2026.
 
**Alumno:** Elias Mereles
 
## Descripción
 
Woof es una app Android que muestra una lista de perros con su foto, nombre, edad y actividad favorita. Al tocar una tarjeta, se expande con una animación para mostrar información adicional (hobby) del perro.
 
## Temas aplicados
 
- **Material Theming con Jetpack Compose**: personalización de colores, tipografía y formas usando Material Design 3 (`Theme.kt`, `Color.kt`, `Type.kt`, `Shape.kt`), soporte de tema claro y oscuro, y agregado de un Top App Bar.
- **Animación simple con Jetpack Compose**: uso de `AnimatedVisibility` y animación tipo `spring()` para expandir/colapsar cada elemento de la lista, junto con el cambio de ícono (`ExpandMore` / `ExpandLess`).
